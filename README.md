# react-native-mpos-wrapper

React-Native wrapper for softPos SDK

## Installation

```sh
npm install react-native-mpos-wrapper
```

## Usage

```js
import { multiply } from "react-native-mpos-wrapper";

// ...

const result = await multiply(3, 7);
```
